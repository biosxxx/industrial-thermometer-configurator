# Changelog

## 0.1.0

- Rebuilt the single-page thermometer prototype as a Vite + React + TypeScript application.
- Extracted thermometer state, catalogs, selectors, and derived logic into typed modules.
- Split the UI into feature forms, widgets, a dedicated SVG preview renderer, and a rich technical editor.
- Added PDF generation, linting, typechecking, unit tests, integration tests, Playwright smoke coverage, and website deployment workflow.
